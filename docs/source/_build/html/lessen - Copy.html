Setting up your PICO¶

Raspberry Pi Pico是一个微型奇迹，将支撑从智能家居系统到工业工厂等一切的技术置于掌中。

无论你是想学习MicroPython编程语言，还是想迈出物理计算的第一步，或者想建立一个硬件项目，Raspberry Pi Pico–及其令人惊叹的社区–将支持你的每一步。
